Return Policy

Standard items

We accept returns on non-custom items if the product is returned in original condition.

  • Item must be unused
  • Item must be in original condition
  • Item must be returned in original packaging when possible

Custom work

Custom items are made to order and are not eligible for return.

This includes items made to custom size, material, finish, design, or personalization.

Return process

  1. Contact us before returning any item
  2. Include your order details
  3. We will provide return instructions

Return shipping

  • Buyer is responsible for return shipping
  • Original shipping costs are non-refundable

Refunds

Refunds are issued after the returned item is received and inspected.

Questions

If you are unsure whether an item qualifies for return, please contact us before purchasing.
